Fendix

DAST + SAST in one PR check. Fails only when both engines confirm.

  • Confirmed findings. The runtime probe and static analysis must agree before fendix fails your build. False-positive flood, gone.
  • Single binary. Drop into CI in 30 seconds. Active probes off by default; enable with one flag when you mean it.
  • No telemetry. Default outbound: only HTTP requests to your scan target. Verify with tcpdump.

Install (macOS & Linux)

curl -fsSL https://get.fendix.dev/install.sh | sh

Downloads the latest release binary for your platform, verifies its sha256 checksum, and installs it to /usr/local/bin/fendix.

First scan

fendix scan --url https://api.example.com --format html --output report.html
open report.html

Detects missing security headers, CORS misconfig, auth bypasses, sensitive data exposure, and rate-limiting gaps — without sending destructive payloads.

Inspect before piping

Sane practice for any curl | sh installer:

curl -fsSL https://get.fendix.dev/install.sh | less

Other install paths

  • Homebrew: brew tap Abdel-RahmanSaied/fendix && brew install fendix
  • Debian/Ubuntu: download the .deb from the latest release
  • RHEL/Fedora: download the .rpm from the same page
  • Docker: docker pull ghcr.io/abdel-rahmansaied/fendix:latest
  • Manual: grab a binary + sha256 from the releases page

Customize

  • FENDIX_DIR=$HOME/.local/bin — install to a user-writable directory
  • FENDIX_VERSION=v0.6.0 — pin a specific version
